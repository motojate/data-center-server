# Branch Rule

| 브랜치 유형 | 작업 내용        | 예시 이름          |
| ----------- | ---------------- | ------------------ |
| feat        | 새로운 기능 개발 | feat/sign-up       |
| bugfix      | 버그 수정        | bugfix/sign-up     |
| hotfix      | 긴급 버그 수정   | hotfix/sign-up     |
| release     | 릴리즈 준비 작업 | release/v1.2       |
| refactor    | 코드 리팩토링    | refactor/common-ui |

# Commit Rule

```
[FEAT] 수정 사항 제목 or 코드
[RV] 리뷰어
- 수정 내역
- 수정 내역
```

# Project Architecture

프레임워크 : NestJS
언어 : 타입스크립트
DB : mongoDB
